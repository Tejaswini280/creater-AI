## Project Architecture
- **Frontend**: React with TypeScript, Wouter routing, TanStack Query, Tailwind CSS
- **Backend**: Express.js with TypeScript, PostgreSQL, Drizzle ORM
- **AI Services**: OpenAI GPT-4o integration for content generation
- **Authentication**: Replit Auth with session management
- **Database**: PostgreSQL with comprehensive schema for users, content, social accounts, metrics
- **External APIs**: YouTube Data API v3 (OAuth ready), Google Trends integration

- **Creator AI Studio**: Complete Google AI Studio clone with Gemini 2.0 Flash, 2M context window, multimodal analysis
